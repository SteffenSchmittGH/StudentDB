◆ read()

void Student::read ( std::istream & in)
virtual

Reads student data from the input stream and populates the student's attributes.

The function reads the student's Matrikelnumber, last name, first name, date of birth (in day.month.year format), and address from the provided input stream. The date of birth is parsed and converted into a Poco::Data::Date object. After reading the basic student information, the student's address is read using the Address::read function.

Parameters
inThe input stream from which to read the student's data

bool Student::removeEnrollment ( unsigned int courseID)

Removes an enrollment from the student's list of enrollments based on the course ID.

Iterates through the student's enrollments and removes the enrollment if the course with the specified courseID is found. If the course is found and removed, the function returns true. Otherwise, it returns false.

Parameters
courseIDThe ID of the course to be removed.
Returns
True if the enrollment was successfully removed; false if the course was not found.

Sets an enrollment for the student in a given course.

Updates an existing enrollment in the student's enrollment list.

This function iterates through the student's enrollments and searches for the course with the specified courseKey. Once the course is found, the corresponding enrollment is updated with the provided new Enrollment object.

Parameters

newEnrollmentThe new enrollment details to replace the existing enrollment for the course.

void Student::write ( std::ostream & out)
virtual

Writes the student's data to the output stream in a formatted manner.

The function writes the student's Matrikelnumber, last name, first name, date of birth, and address to the provided output stream. The data is separated by semicolons and the date is formatted as day.month.year. The student's address is written using the Address class's write function.

Parameters
outThe output stream to write the student's data to
