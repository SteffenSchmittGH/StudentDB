◆ connectWithServer()

void SimpleUI::connectWithServer ( )

Connects to the HDA server and adds students to the database.

This function establishes a connection to the server at "www.hhs.users.h-da.cloud" on port "4242" to fetch student data in JSON format. It prompts the user for the number of students to add, then repeatedly sends a "generate" command to the server to retrieve student information. The extracted data is then used to create Student objects, which are adde to the student database. If an error occurs while adding a student, an error message is displayed.
